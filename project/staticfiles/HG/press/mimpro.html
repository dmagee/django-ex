HeteroGenius Medical Image Manager (HG-MIM) Pro Released

1st January, 2016

HeteroGenius Medical Image Manager (HG-MIM) Pro is now released. HG-MIM Pro is a fully supported web platform for the management and analysis of digital histopathology images from whole slide scanners. In addition to professional support, HG-MIM Pro adds extra functionality to the basic (free) HG-MIM platform including; Automatic annotation of tissue pieces, Stereology, Barcode reading, Colour based quantitation, Flexible Report Generation and Export.
